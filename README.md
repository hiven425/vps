# VPS安全加固脚本

全面的VPS安全加固和代理服务搭建解决方案，集成安全监控、可视化状态显示和智能管理功能。

## ✨ 主要特性

### 🛡️ 安全加固功能
- **系统更新和用户管理**: 自动更新系统，创建安全用户
- **SSH安全配置**: 端口修改、密钥认证、安全参数优化
- **防火墙配置**: UFW防火墙规则设置和管理
- **fail2ban防护**: 入侵检测和自动封禁系统
- **网络安全配置**: 系统内核参数优化
- **系统监控和清理**: 性能监控和垃圾清理

### 🚀 代理服务功能
- **证书管理**: Cloudflare DNS自动申请SSL证书
- **Hysteria2服务**: 高性能代理协议部署
- **X-UI面板**: 可视化代理管理界面
- **Sub-Store服务**: 订阅转换和管理
- **Nginx分流配置**: 智能流量分发
- **vless+reality代理**: 最新伪装技术

### 🎨 增强体验
- **可视化状态显示**: 🟢🟡🔴 直观的服务状态指示器
- **安全状态概览**: 一键查看系统整体安全状况
- **智能fail2ban管理**: 增强的状态显示和管理功能
- **交互式配置**: 友好的用户界面和操作提示

## 🚀 快速开始

### 基本使用
```bash
# 下载脚本
chmod +x security-hardening.sh

# 运行脚本
./security-hardening.sh

# 查看安全状态概览
选择选项 0

# 一键安全加固
选择选项 21
```

## 🔧 系统要求

- **操作系统**: Debian/Ubuntu
- **权限**: root用户
- **依赖**: fail2ban, ufw (可选)
