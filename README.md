# VPS安全加固脚本

全面的VPS安全加固和代理服务搭建解决方案，集成安全监控、可视化状态显示和智能管理功能。

## ✨ 主要特性

### 🛡️ 安全加固功能
- **系统更新和用户管理**: 自动更新系统，创建安全用户
- **SSH安全配置**: 端口修改、密钥认证、安全参数优化
- **防火墙配置**: UFW防火墙规则设置和管理
- **fail2ban防护**: 入侵检测和自动封禁系统
- **网络安全配置**: 系统内核参数优化
- **系统监控和清理**: 性能监控和垃圾清理

### 🚀 代理服务功能
- **证书管理**: Cloudflare DNS自动申请SSL证书
- **Hysteria2服务**: 高性能代理协议部署
- **X-UI面板**: 可视化代理管理界面
- **Sub-Store服务**: 订阅转换和管理
- **Nginx分流配置**: 智能流量分发
- **vless+reality代理**: 最新伪装技术

### 🎨 增强体验
- **可视化状态显示**: 🟢🟡🔴 直观的服务状态指示器
- **安全状态概览**: 一键查看系统整体安全状况
- **智能fail2ban管理**: 增强的状态显示和管理功能
- **交互式配置**: 友好的用户界面和操作提示

## 🚀 快速开始

## 🔧 系统要求

- **操作系统**: Debian/Ubuntu系统
- **权限**: 必须使用root用户运行
- **网络**: 服务器需要能够访问外网

## ⚠️ 重要安全提示

### 🔐 SSH安全配置
- **默认SSH端口**: 脚本建议使用4025端口（而非22）
- **密钥认证**: 禁用密码认证前会验证SSH密钥配置
- **用户验证**: 交互式配置允许登录的用户，避免被锁定
- **安全重启**: SSH服务重启前会进行多重安全检查

### 🛡️ 防火墙配置
- 自动检测SSH端口并配置防火墙规则
- 智能检测Web服务并推荐端口配置
- 配置顺序确保SSH服务正常后再启用防火墙

## 💡 使用建议

### 首次使用
1. **查看状态**: 先选择选项 0 查看当前安全状态
2. **SSH配置**: 建议先单独配置SSH（选项4）并测试连接
3. **一键加固**: 确认SSH正常后选择选项 21 进行全面加固

### 安全操作
- **保持连接**: SSH配置期间保持当前连接，新终端测试
- **备份配置**: 脚本自动备份重要配置文件
- **逐步配置**: 建议逐项配置而非一键执行（首次使用）

### 代理服务
如需搭建代理服务：
1. 先完成基础安全加固
2. 准备Cloudflare相关信息（域名、API Token等）
3. 按需选择代理协议（Hysteria2、X-UI等）
