# VPS管理脚本

全面的VPS安全加固和代理服务管理脚本，一次安装，快捷键控制。

## 🚀 快速开始

```bash
bash <(curl -L https://raw.githubusercontent.com/hiven425/vps/refs/heads/main/vps.sh)
```

安装后可使用快捷键：
```bash
vps  # 快速启动管理界面
```

## ✨ 主要功能

### 🛡️ 安全加固模块
- **SSH安全配置**: 端口修改、密钥认证、防锁定机制
- **防火墙配置**: UFW防火墙规则设置和管理  
- **fail2ban防护**: 入侵检测和自动封禁系统
- **系统更新**: 自动更新和用户管理
- **安全扫描**: 系统漏洞检测和安全评估
- **一键加固**: 自动化安全配置

### 🚀 代理服务模块

- **sing-box**: 现代化高性能代理工具
- **Hysteria2服务**: 高性能UDP代理协议
- **vless+reality代理**: 最新伪装技术
- **证书管理**: Cloudflare API自动申请SSL证书
- **Nginx分流**: 智能流量分发配置

### 🎨 增强体验
- **可视化状态**: 🟢🟡🔴 直观的服务状态指示器
- **安全概览**: 一键查看系统整体安全状况
- **智能管理**: 增强的fail2ban状态显示和管理

## 💡 使用建议

### 首次使用
1. **查看状态**: 先选择选项 0 查看当前安全状态
2. **SSH配置**: 建议先单独配置SSH（选项4）并测试连接
3. **一键加固**: 确认SSH正常后选择选项 14 进行全面安全加固

### 模块化使用
- **安全加固模块**: 选项1-14，专注于系统安全配置
- **代理服务模块**: 选项20-27，专注于代理服务部署
- **独立运行**: 两个模块可以独立使用，互不干扰

### 安全操作
- **保持连接**: SSH配置期间保持当前连接，新终端测试
- **备份配置**: 脚本自动备份重要配置文件
- **逐步配置**: 建议逐项配置而非一键执行（首次使用）

## 🔒 安全特性

### SSH安全配置
- **默认端口**: 建议使用4025端口（而非22）
- **密钥认证**: 禁用密码认证前会验证SSH密钥配置
- **用户验证**: 交互式配置允许登录的用户，避免被锁定
- **连接保活**: 60秒心跳，防止超时断开
- **性能优化**: 禁用DNS反向解析，加快登录速度

### 防火墙配置
- 自动检测SSH端口并配置防火墙规则
- 智能检测Web服务并推荐端口配置
- 配置顺序确保SSH服务正常后再启用防火墙

### fail2ban防护
- SSH暴力破解防护
- 自动IP封禁和解封
- 可视化状态监控
- 详细日志分析
